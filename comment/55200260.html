CÂU CẢM THÁN:

  • What + a/an + Adj + N! : What a nice dress!

  • How + Adj + the + N + be/V : How nice the dress is!

MỘT SỐ TỪ BASIC CẦN NHỚ:

  • Even though = although = though : mặc dù ...

  • As long as = so long as = provided that = proving that : miễn là ...

  • If - clause dùng Tobe vs mọi S

  • If clause = If it hadn't been for= But for = Without

  • If = suppose : giả sử

CÂU ĐIỀU KIỆN:

A/ Loại 1 : Diễn tả đkien có thể xảy ra ở htai/ tg lai

<pre> If + HTĐ , S + will +O + V </pre>

B/ Loại 2 : diễn tả đkien không thể xảy ra ở hiện tại

<pre> If + QKĐ , S + would + V + O </pre>

C/ Loại 3 : diễn tả đkien không thể xảy ra ở qkhu

<pre> If + ( S + had + P2) , S would have P2 + O </pre>
