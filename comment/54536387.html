Từ vựng tiếng Anh, thuật ngữ về Công nghệ hay được sử dụng.

  1. Alphanumeric data /ˌæl.fə.njuːˈmer.ɪk deɪ.tə/ Dữ liệu chữ số, dữ liệu cấu thành các chữ cái và bất kỳ chữ số từ 0 đến 9

  2. Available /əˈveɪləbl/ Dùng được, có hiệu lực

  3. Application /ˌæplɪˈkeɪʃn/ Ứng dụng

  4. Alphabetical catalog /ˌæl.fəˈbet.ɪ.kəl kæt̬.əl.ɑːɡ/ Mục lục xếp theo trật tự chữ cái

  5. Abbreviation /əˌbriːviˈeɪʃn/ Sự tóm tắt, rút gọn

  6. Analysis /əˈnæləsɪs/ Phân tích

  7. Broad classification /brɔːd ˌklæs.ɪ.fɪˈkeɪ.ʃən/ Phân loại tổng quát

  8. Background /ˈbækɡraʊnd/ Bối cảnh, bổ trợ

  9. Cluster controller /ˈklʌs.tər kənˈtrəʊ.lər/ Bộ điều khiển trùm

  10. Chief source of information: Nguồn thông tin chính. Dùng tạo ra phần mô tả của một biểu ghi thay thế như trang bìa sách, hình nhan đề phim hoạt hình hay nhãn băng đĩa

  11. Configuration /kənˌfɪɡəˈreɪʃn/ Cấu hình

  12. Common /ˈkɒmən/ Thông thường

  13. Compatible /kəmˈpætəbl/ Tương thích

  14. Data /ˈdeɪtə/ Dữ liệu

  15. Gateway /ˈɡeɪtweɪ/ Cổng kết nối Internet cho những mạng lớn

  16. Intranet /ˈɪntrənet/ Mạng nội bộ

  17. Graphics /ˈɡræfɪks/ Đồ họa

  18. Multi-user /mʌl.ti-juː.zər/ Đa người dùng

  19. Operating system /ˈɒp.ər.eɪ.tɪŋ ˌsɪs.təm/ Hệ điều hành

  20. OSI (Open System Interconnection)Mô hình chuẩn OSI

  21. Operation /ɒpəˈreɪʃn/ Thao tác

  22. Packet /ˈpæk.ɪt/ Gói dữ liệu

  23. PPP (Point-to-Point Protocol)Một giao thức kết nối Internet tin cậy thông qua Modem

  24. Port /pɔːt/ Cổng

  25. Protocol /ˈprəʊtəkɒl/ Giao thức

  26. Pinpoint /ˈpɪnpɔɪnt/ Chỉ ra một cách chính xác

  27. Source Code /ˈsɔːs ˌkəʊd/ Mã nguồn của một file hay một chương trình

  28. Storage /ˈstɔːrɪdʒ/ Lưu trữ

  29. Technical /ˈteknɪkl/ Thuộc về kỹ thuật

  30. Text /tekst/ Văn bản chỉ bao gồm ký tự

  31. Remote Access /rɪˈməʊt ˈæk.ses/ Truy cập từ xa qua mạng

  32. Union catalog /ˈjuː.nj.ən kæt̬.əl.ɑːɡ/ Mục lục liên hợp. Thư mục thể hiện những tài liệu ở nhiều thư viện hay kho tư liệu
