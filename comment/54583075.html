LOẠI 1: Diễn tả đkien k thể xảy ra ở HT/TL.

<pre> If + HTĐ, S + will + V + O </pre>
  • Loại O : If + HTĐ, HTĐ ( diễn tả điều hiển nhiên)

If + HTĐ, V + O (diễn tả mệnh lệnh)

  • If = unless : trừ khi...nếu k thì....

  • As long as/ so long as/ provided that/ proving that : miễn là

  • Will = can = should = may = might

LOẠI 2 : Không thể xảy ra ở HT

<pre> If + QKĐ, S + would + V + O </pre>
eg: if i earned enough for leaving, i would give you.

  • If = suppose : giả sử

  • If clause dùng tobe WERE vs mọi S.

  • would = could

LOẠI 3 : Không thể xảy ra ở QK

<pre> If + S + had + P2, S would have P2 + O </pre>
  • If clause = without/ but for/ if it hadn't been for + N (S + would have P2) : nếu k có thì
eg : if he hadn't helf us, we couldn't finished it.
=> without his helf, we couldn't have finished.
